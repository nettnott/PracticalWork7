# Фибоначчи

В этом задании мы использовали следующие способы отладки:
- Установка обычных точек останова.
- Запуск приложения в режиме отладки.
- Step Over (пошаговое выполнение без захода в методы) для прохождения через функцию Fibonacci и цикл for.
- Окно «Локальные» (Locals) для просмотра значений переменных (n, n1, n2, sum, i).
- Кнопка «Продолжить» (Continue) для перехода от одной точки останова к следующей.
- Удаление точек останова.
- Перезапуск отладки (Restart Debugging) после исправления кода.
- Повторный запуск отладки для проверки исправления ошибки в цикле.
