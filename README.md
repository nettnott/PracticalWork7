# Галактики

В этом задании мы использовали следующие способы отладки:
- Запуск приложения в режиме отладки с помощью клавиши F5.
- Установку и удаление точек останова (F9 и щелчком в поле рядом со строкой кода).
- Перезапуск отладки (Ctrl + Shift + F5).
- Пошаговое выполнение: Выполнение по одной инструкции за раз (F11) и Step Over (F10).
- Просмотр значений переменных.
- Горячую перезагрузку.
- Изменение и продолжение.
