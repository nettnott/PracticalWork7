# Буквы

В этом задании мы использовали следующие способы отладки:
- Установку точек останова.
- Запуск отладки (F5).
- Остановка отладки (Shift + F5).
- Пошаговое выполнение: Шаг с обходом (F10), Шаг в (F11), Шаг выхода (Shift + F11).
- «Выполнить до щелчка». (Run to click).
- Окно «Локальные» (Locals) (Ctrl + D, L).
- Окно «Авто» (Autos) (Ctrl + D, A).
- Окно «Просмотр» (Watch).
- Окно «Стек вызовов» (Ctrl + D, C).
- Просмотр значений переменных.
