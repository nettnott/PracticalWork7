# Буквы

В этом задании мы использовали следующие способы отладки:
- Установку точек останова щелчком в левом поле.
- Запуск отладки (F5).
- Пошаговое выполнение: Step Over (F10), Step Into (F11), Step Out (Shift + F11)
- Окно «Локальные» (Locals) (Ctrl + D, L).
- Окно «Авто» (Autos) (Ctrl + D, A).
- Окно «Просмотр» (Watch) — добавление переменных для наблюдения.
- Окно «Стек вызовов» (Call Stack) (Ctrl + D, C).
- DataTips - просмотр значений переменных при наведении мыши (включая развёртывание массивов).
- Выполнить до курсора (Run to Click) (Ctrl + F10 или кнопка при наведении).
- Перезапуск отладки (Ctrl + Shift + F5).
- Остановка отладки (Shift + F5).
