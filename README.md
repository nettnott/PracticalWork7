# Галактики

В этом задании мы использовали следующие способы отладки:
- Запуск приложения в режиме отладки с помощью клавиши F5.
- Установку и удаление точек останова (F9 и щелчком в поле).
- Перезапуск отладки (Ctrl + Shift + F5).
- Пошаговое выполнение: Step Into (F11) и Step Over (F10).
- Просмотр значений переменных с помощью DataTips (подсказки при наведении мыши).
- Окно Стек вызовов (Call Stack).
- Горячую перезагрузку / Изменение и продолжение (Edit and Continue) - редактирование кода во время отладки.
- Обработку исключений через Помощник по исключениям (Exception Assistant).
- Точки трассировки (Tracepoints) для записи сообщений в окно вывода.
