# Фибоначчи

В этом задании мы использовали следующие способы отладки:
- Установка точек останова и пошаговое выполнение.
- Step Over для перехода в цикл for.
- Запуск приложения в режиме отладки.
- Окно «Локальные» для просмотра значений переменных.
- Кнопка «Продолжить» для перехода от одной точки останова к следующей.
- Удаление точек останова.
- Перезапуск отладки после исправления кода.
- Повторный запуск отладки для проверки исправления ошибки в цикле.
